## AIM

To demonstrate the client/server communication for sending and receiving a message using MQTT Broker. 

## APPARATUS REQUIRED

- Espressif ESP 32 dev module
- Mosquitto MQTT Server

## PRE-CONDITIONS

- Mosquitto MQTT broker running and reachable on the network.
- ESP32 flashed with correct Wi-Fi SSID and password in the code.
- MQTT topics used by client and server are agreed and not blocked by broker.

## POST-CONDITIONS

- MQTT client successfully publishes a test message and server receives it.
- Successful verification using MQTT client (e.g., mosquitto_sub) or serial monitor.
