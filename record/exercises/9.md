## AIM

To simulate smart temperature and humidity monitoring using MQTT Broker.

## APPARATUS REQUIRED

- Espressif ESP32 Dev Module.
- DHT22 (Temperature and Humidity Sensor).

## PRE-CONDITIONS

- Mosquitto MQTT broker running and reachable on the network.
- ESP32 devices flashed with Wi‑Fi credentials.
- DHT22 sensor connected correctly (VCC, GND, data pin) with pull-up if required.

## POST-CONDITIONS

- Server publishes periodic temperature and humidity readings to MQTT topics.
- Client subscribes and successfully receives and displays sensor data; verify values are within expected ranges.
