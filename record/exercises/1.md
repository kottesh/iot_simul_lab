## AIM

To build an exam alarm system that will be able to beep every hour, half an hour, or with custom time interval.

## APPARATUS REQUIRED

- Arduino Uno
- Breadboard
- RTC DS1307 module
- I2C LCD Display (16x2)
- Pushbuttons (x4) (Mode, Up, Down, Set)
- Buzzer
- Jumper wires

## PRE-CONDITIONS

- Hardware components are wired correctly, ensuring the RTC module and I2C LCD are securely connected to the Arduino I2C pins (A4, A5).
- Required libraries (`Wire.h`, `RTClib.h`, `LiquidCrystal_I2C.h`) are installed in the Arduino IDE.
- The RTC DS1307 module is properly configured with the current date and time.
- The code is successfully compiled and uploaded to the Arduino Uno board.

## POST-CONDITIONS

- The LCD correctly displays the current time, elapsed/remaining time, and configuration menus.
- The user can configure alarm intervals (half-hour, one-hour, custom, or random) using the pushbuttons.
- The buzzer sounds alerts automatically at the configured intervals while the exam is running.
- The exam process starts and stops gracefully upon interacting with the Set button.
