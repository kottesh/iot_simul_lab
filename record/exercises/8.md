## AIM

To perform simple arithmetic operations for two integer numbers using MQTT Broker.

## APPARATUS REQUIRED

- Espressif ESP 32 Dev Module
- ILI9341 Cap Touch Display

## PRE-CONDITIONS

- Mosquitto MQTT broker running and reachable on the network.
- Both ESP32 devices (server and client) programmed with correct Wi‑Fi credentials.
- Topics for arithmetic request/response established and unique.

## POST-CONDITIONS

- Server receives operands from client, performs arithmetic operations and publishes results.
- Client receives and displays the computed result; verify via serial monitor or broker.
