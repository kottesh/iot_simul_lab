## AIM

To build a Smart Four Wheeler Vehicle Parking System using MQTT Broker.

## APPARATUS REQUIRED

- Espressif ESP32 Dev Module
- HC-SR04 Proximity Sensor
- ILI9341 Cap Touch LCD Display

## PRE-CONDITIONS

- Mosquitto MQTT broker running and reachable on the network.
- ESP32 devices flashed with correct Wi‑Fi credentials.
- HC-SR04 wired correctly (Trig, Echo, VCC, GND) with proper voltage level handling.
- ILI9341 display wired and library initialized (if used on client).

## POST-CONDITIONS

- Server detects vehicle presence/distance and publishes parking slot status over MQTT.
- Client receives parking updates and displays availability on the LCD; verify ultrasonic readings and displayed status match physical test.
