## AIM

To simulate the Music Visualizer

## APPARATUS REQUIRED

- Arduino Uno
- MAX7219 8x8 LED Matrix Module
- 10-Segment LED Bar Graph
- Buzzer
- Resistors (x10)
- Pushbuttons (x2) (Next, Previous)
- Jumper wires

## PRE-CONDITIONS

- The LED matrix is correctly connected via SPI (DIN, CS, CLK).
- The 10-Segment LED Bar Graph is properly wired with current-limiting resistors to the Arduino digital and analog pins.
- The buzzer and pushbuttons are appropriately connected.
- The `LedControl.h` library is installed in the Arduino IDE.
- The code is successfully compiled and uploaded to the Arduino Uno board.

## POST-CONDITIONS

- The buzzer plays melodies based on predefined musical notes and duration.
- The LED Bar Graph lights up proportionally to the frequency of the current note being played.
- The 8x8 LED Matrix visualizes the audio frequencies using dynamic bar heights and simulated harmonics.
- The pushbuttons allow the user to smoothly switch between different songs in the playlist.
