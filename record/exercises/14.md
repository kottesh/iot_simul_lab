## AIM

To Develop Tic Tac Toe Game using MQTT Broker.

## APPARATUS REQUIRED

- ESP 32 Dev Module

## PRE-CONDITIONS

- Mosquitto MQTT broker running and reachable on the network.
- ESP32 devices flashed with Wi‑Fi credentials for both players (if using two devices).
- Input method (touch/display or buttons) configured and tested.

## POST-CONDITIONS

- Game moves are exchanged over MQTT and board state stays synchronized between players.
- Game correctly detects win/draw conditions and publishes final result; verify by playing a full game and observing MQTT messages/serial output.
