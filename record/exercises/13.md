## AIM

To simulate Bank Locker Access System and send the SMS notification to the Locker-Hirer and give the alarm for detecting the unauthorized person using MQTT Broker.

## APPARATUS REQUIRED

- Espressif ESP32 Dev Module
- Breadboard
- 4x LED
- 2x Servo Motor
- Keypad
- LCD I2C Display
- Buzzer

## PRE-CONDITIONS

- Mosquitto MQTT broker running and reachable on the network.
- ESP32 flashed with Wi‑Fi credentials.
- Keypad, servos, LEDs, LCD and buzzer wired and libraries included.
- Ensure servo power supply can provide required current (use separate supply if needed).

## POST-CONDITIONS

- Authorized access via keypad unlocks locker (servo movement) and publishes access event to MQTT.
- Unauthorized attempts trigger alarm, MQTT alert and SMS notification if configured; verify via serial monitor and MQTT topics.
