## AIM

To simulate electronic road passing system and toll collection without human intervention and send SMS to the vehicle owner using MQTT Broker.

## APPARATUS REQUIRED

- Espressif ESP32 Dev Module
- MFRC522 RFID Scanner
- SSD1306 OLED 128X64
- Servo Motor
- 3xLED
- Buzzer


## PRE-CONDITIONS

- Mosquitto MQTT broker running and reachable on the network.
- ESP32 flashed with Wi‑Fi credentials.
- MFRC522 RFID module connected (SDA, SCK, MOSI, MISO, RST) and library configured.
- GSM/SMS gateway or service configured if SMS sending is included (or test using serial/logging).

## POST-CONDITIONS

- Authorized RFID scans allow passage; system publishes events to MQTT broker.
- On unauthorized access, alarm and notification are triggered and published; verify via serial monitor and MQTT topic messages.
